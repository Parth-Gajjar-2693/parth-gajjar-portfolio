## 📂 Project Setup

### 1. Clone Repository

```bash
git clone https://github.com/your-username/your-portfolio.git
```

### 2. Navigate to Project

```bash
cd your-portfolio
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Run Development Server

```bash
npm run dev
```

App will run at:

```
http://localhost:5173/
```
